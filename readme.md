# Merry Christmas (Unfinished)

### 目录结构

* config --配置文件主要为需要加载的img列表
* dist  --打包后的文件
* img  --img文件
* js --img文件
* node_modules 
* index.html --主页面
* webpack.config.js

### 快速预览
直接打开index.html即可预览
或者访问[这个网站](https://newnewking.github.io/MerryChristmas/)

### 开发者调试
命令行进入该目录 输入webpack
然后打开index.html预览
